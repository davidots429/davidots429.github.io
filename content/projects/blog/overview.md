# GitHub Pages 블로그 개요

이 프로젝트는 Next.js와 Tailwind CSS를 사용하여 GitHub Pages에서 호스팅되는 개인 블로그입니다.

## 주요 기능

- 마크다운 기반 콘텐츠 관리
- 동적 라우팅
- 자동 배포 (GitHub Actions)
- 반응형 디자인

## 기술 스택

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS v4
- **Deployment**: GitHub Pages
