# 지식교류 게시판

게시판 구현 프로젝트

## 개발 환경

 - **Html**, **JavaScript** : UI/UX
 - **Java** : Server
 - **CSS**, **Bootstrap** : Page Design
 - **MariaDB** : DBMS
 - **Apache Tomcat** : Runtime
 - **Eclipse IDE** : Development tool

## 개요
가장 간단한 구조의 웹 사이트를 제작해 봄으로써 HTML/CSS 및 Bootstrap으로 페이지의 UI 및 디자인을 설계해보고 DBMS 및 SQL 사용 경험을 가져보고자 했다.
