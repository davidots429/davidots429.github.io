# WineReco

품종 기반 와인 추천 모델을 사용한 와인 추천 웹 사이트

## 개발 환경

 - **React**, **JavaScript** : Front-end
 - **Python** : Back-end
 - **PostgreSQL** : DBMS
 - **Flask**, **npm** : runtime
 - **Tailwindcss** : Design
  
## 개요
만들어 낸 [품종 기반 와인 추천 모델](/projects/data/20250430_winebertmodel)을 한 번 서비스화 시켜보고 싶었다. 또한 React와 PostgreSQL 사용 경험을 갖고자 했다.
