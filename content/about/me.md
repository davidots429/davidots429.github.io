# 안녕하세요!

저는 웹 개발을 열정적으로 공부하고 있는 개발자입니다.

## 기술 스택

### Frontend

- **JavaScript/TypeScript**: 언어
- **React**: UI 라이브러리
- **Next.js**: React 풀스택 프레임워크
- **Tailwind CSS**: 스타일링

### Backend

- **Node.js**: 런타임
- **Express**: 웹 프레임워크

### Tools & Others

- **Git/GitHub**: 버전 관리
- **VS Code**: 개발 에디터
- **Linux**: 개발 환경

## 경험

- 웹 개발의 기초부터 심화까지

- 개인 프로젝트를 통한 실무 경험
